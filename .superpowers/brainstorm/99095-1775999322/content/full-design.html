Design Decisions

  • My Models = card grid (2-per-row) — downloaded + remote, unified. Each card shows source (green dot + name), model name, size/quant, tok/s, and capability badge.
  • "+ Add Server" button in the My Models section header — not a separate tab or screen.
  • Browse HF = list format (not cards) — browse results are denser and benefit from list layout. Shows model name, publisher, param count, download count, and compatibility badge.
  • Compatibility badges are front-and-center — green "Runs Well", yellow "Runs Slow", red "Too Large" on every HF browse result. This is the core differentiator.
  • Tap a My Models card → switch to Chat tab with that model loaded.
  • Tap a Browse result → detail view with download options and full compatibility info.
  • Gear icon top-right → settings sheet (inference defaults, about, future prefs).
  • Offline models dimmed at 45% opacity with red "offline" badge.
