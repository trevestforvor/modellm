# Plan 05-02: Inference Parameters UI — ChatSettingsView with Presets and Sliders

## Goal

Extend `ChatSettingsView` (established in Phase 4 with system prompt presets) with temperature and top-p controls. Add preset pills ("Precise", "Balanced", "Creative") that snap sliders to preset values. Wire new `InferenceParams` fields through `LlamaSession`'s sampler chain so parameters take effect on the next generation call — without reloading the model from disk.

## Must-Haves

- [ ] `ChatSettingsView` has preset pill row: "Precise" / "Balanced" / "Creative" — tapping writes temperature + topP to `DownloadedModel` in SwiftData
- [ ] Selected preset pill shows `#8B7CF0` fill; unselected shows `#302E42` border
- [ ] "Advanced" disclosure group below presets shows Temperature slider (0.0–2.0, step 0.1) and Top-P slider (0.0–1.0, step 0.05)
- [ ] Sliders have `#8B7CF0` tint, SF Mono value label right-aligned, min/max labels in `#6B6980`
- [ ] Slider changes write to `DownloadedModel.temperature` / `topP` in SwiftData immediately
- [ ] System prompt section preserved from Phase 4 (preset chips + editable field)
- [ ] Settings sections use glass material backgrounds (`#1A1830` at 60% opacity, `#302E42` border, 12pt corner radius)
- [ ] `LlamaSession` builds a sampler chain from `InferenceParams.temperature` and `topP` — called per `generate()` invocation (not at init)
- [ ] `InferenceService.generate(prompt:params:)` accepts updated `InferenceParams` — no model reload required to change sampling

## Context

Phase 4 established `ChatSettingsView` with a gear icon in the chat nav bar, system prompt presets, and an editable text field. `LlamaSession` wraps the llama.cpp context. In Phase 4, `InferenceParams` had no sampling fields — those are added in Plan 05-01. Phase 5 extends both the UI and the llama.cpp session to use the new fields. The key architectural point: the sampler chain is built per `generate()` call (not at `LlamaSession` init), so changing temperature doesn't require reloading the 2-4GB model from disk.

Add a method `buildSamplerChain(params: InferenceParams) -> OpaquePointer` (return type matches whatever the XCFramework exposes for `llama_sampler*`):

```swift
/// Build a sampler chain from InferenceParams.
/// Called at the start of each generate() call — NOT at init.
/// This allows temperature/top-p changes without model reload.
func buildSamplerChain(params: InferenceParams) -> OpaquePointer {
    var sparams = llama_sampler_chain_default_params()
    let chain = llama_sampler_chain_init(sparams)!
    // top-p first (filters the distribution), then temperature (scales it)
    llama_sampler_chain_add(chain, llama_sampler_init_top_p(params.topP, 1))
    llama_sampler_chain_add(chain, llama_sampler_init_temp(params.temperature))
    llama_sampler_chain_add(chain, llama_sampler_init_dist(LLAMA_DEFAULT_SEED))
    return chain
}
```

Update the decode loop in `generate()` to:
1. Call `buildSamplerChain(params:)` at the start of generation
2. Use `llama_sampler_sample(chain, ctx, -1)` to get the next token
3. Call `llama_sampler_free(chain)` in a `defer` block

Also update `generate()` (or the method signature called by `InferenceService`) to accept `params: InferenceParams` so callers can pass fresh params per invocation.

IMPORTANT: After reading the actual XCFramework headers (from Phase 4 Task 04-01-02), verify the exact function signatures. The pattern above matches llama.cpp b5046+ API. If the XCFramework exposes an older API (`llama_context_params.temp`), use that instead and note the difference in a comment.

```swift
/// Returns a stream of token strings. Runs inference on Task.detached.
/// Pass current model params so temperature/top-p changes take effect
/// without reloading the model from disk.
func generate(prompt: String, params: InferenceParams) -> AsyncThrowingStream<String, Error> {
    AsyncThrowingStream { [weak self] continuation in
        guard let self else {
            continuation.finish(throwing: InferenceError.noActiveSession)
            return
        }
        Task.detached(priority: .userInitiated) {
            guard let session = await self.session else {
                continuation.finish(throwing: InferenceError.noActiveSession)
                return
            }
            await self.setNotCancelled()
            do {
                // Build fresh sampler chain with current params
                let chain = session.buildSamplerChain(params: params)
                defer { llama_sampler_free(chain) }

                // Tokenize and decode loop (existing pattern)
                // ...
                continuation.finish()
            } catch {
                continuation.finish(throwing: error)
            }
        }
    }
}
```

```swift
// MARK: - Preset Pills
private var presetSection: some View {
    VStack(alignment: .leading, spacing: 8) {
        Text("STYLE")
            .font(.system(size: 12, weight: .semibold))
            .tracking(1.2)
            .foregroundStyle(Color(hex: "#9896B0"))

        HStack(spacing: 8) {
            ForEach(InferencePreset.allCases) { preset in
                let isSelected = isPresetActive(preset)
                Button(preset.rawValue) {
                    preset.apply(to: model)
                    // SwiftData auto-saves on next run loop
                }
                .font(.system(size: 14, weight: .medium))
                .foregroundStyle(isSelected ? .white : Color(hex: "#9896B0"))
                .padding(.horizontal, 16)
                .padding(.vertical, 8)
                .background(
                    Capsule()
                        .fill(isSelected ? Color(hex: "#8B7CF0") : Color.clear)
                        .overlay(
                            Capsule()
                                .strokeBorder(
                                    isSelected ? Color.clear : Color(hex: "#302E42"),
                                    lineWidth: 1
                                )
                        )
                )
            }
        }
    }
    .padding()
    .background(
        RoundedRectangle(cornerRadius: 12)
            .fill(Color(hex: "#1A1830").opacity(0.6))
            .overlay(
                RoundedRectangle(cornerRadius: 12)
                    .strokeBorder(Color(hex: "#302E42"), lineWidth: 0.5)
            )
    )
}

private func isPresetActive(_ preset: InferencePreset) -> Bool {
    abs(model.temperature - preset.temperature) < 0.01 &&
    abs(model.topP - preset.topP) < 0.01
}
```
