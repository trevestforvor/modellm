# Plan 05-04: Welcome Screen and V1 Polish

## Goal

Add the first-launch welcome screen with two paths ("Show Me Around" guided download, "Get Started" skip to Browse), gated by `@AppStorage`. The welcome screen is a full MeshGradient poster — no cards, no decoration. Guided path auto-picks the best model for the user's device. After this plan, the v1 pipeline is complete: first-time users land on the welcome screen; returning users go straight to their conversations.

## Must-Haves

- [ ] `WelcomeView` is a full MeshGradient poster: app icon (80pt), "ModelRunner" (28pt bold), tagline (20pt semibold), subtitle (15pt secondary)
- [ ] Two buttons stacked vertically: "Show Me Around" (`#8B7CF0` filled) and "Get Started" (glass material)
- [ ] `ModelRunnerApp` gates on `@AppStorage("hasCompletedOnboarding")` — welcome screen shows on first launch only
- [ ] Tapping either button sets `hasCompletedOnboarding = true` and transitions to `ContentView`
- [ ] Guided path ("Show Me Around"): app filters `DownloadedModel` for `Runs Well` compatibility, picks smallest by file size, sets it as active model and navigates to Chat tab
- [ ] If no downloaded models exist on guided path, fall back to Browse tab (same as "Get Started")
- [ ] "Get Started": navigates to Browse tab with no guided flow
- [ ] Welcome screen has NO cards, NO illustration, NO emoji — the MeshGradient is the visual

## Context

The welcome screen inserts before `ContentView` at the `WindowGroup` level. `ModelContainer` is already initialized on the `WindowGroup` modifier (from Plan 05-01), so `WelcomeView` can access SwiftData to query downloaded models for the guided path. The guided path's best-model picker filters `DownloadedModel` records by compatibility rating and sorts by `fileSizeBytes` ascending.

Design reference: `DESIGN.md` "Welcome Screen (Phase 5)" section. The `MeshGradient` background is the same gradient used across all other screens — reuse the same pattern from `ChatLibraryPreview.playground` or the existing `BrowseView`/`LibraryView` background implementation.

struct WelcomeView: View {
    @Query private var downloadedModels: [DownloadedModel]
    @Environment(\.appContainer) private var appContainer  // or however AppContainer is injected

    let onComplete: (WelcomePath) -> Void

    enum WelcomePath {

        case browse
    }

    // Animate gradient

    var body: some View {

                }

                    .foregroundStyle(.primary)

                Spacer().frame(height: 8)

                // Tagline
                Text("Run AI on your iPhone")
                    .font(.system(size: 20, weight: .semibold))
                    .foregroundStyle(.primary)

                Spacer().frame(height: 8)

                // Subtitle
                Text("Download open-source models. Chat completely offline.")
                    .font(.system(size: 15))
                    .foregroundStyle(Color(hex: "#9896B0"))
                    .multilineTextAlignment(.center)
                    .padding(.horizontal, 32)

                Spacer()

                        Text("Show Me Around")
                            .font(.system(size: 16, weight: .semibold))
                            .foregroundStyle(.white)
                            .frame(maxWidth: .infinity)
                            .frame(height: 52)
                            .background(
                                RoundedRectangle(cornerRadius: 14)
                                    .fill(Color(hex: "#8B7CF0"))
                            )
                    }

                    // Secondary: Get Started
                    Button(action: { onComplete(.browse) }) {
                        Text("Get Started")
                            .font(.system(size: 16, weight: .medium))
                            .foregroundStyle(.primary)
                            .frame(maxWidth: .infinity)
                            .frame(height: 52)
                            .background(
                                RoundedRectangle(cornerRadius: 14)
                                    .fill(Color(hex: "#1A1830").opacity(0.6))
                                    .overlay(
                                        RoundedRectangle(cornerRadius: 14)
                                            .strokeBorder(Color(hex: "#302E42"), lineWidth: 0.5)
                                    )
                            )
                    }
                }
                .padding(.horizontal, 16)
                .padding(.bottom, 48)
            }
        }
        .ignoresSafeArea()
        .preferredColorScheme(.dark)
    }

    // MARK: - Guided path logic

    private func handleShowMeAround() {
        let bestModel = pickBestModel()
        onComplete(.guided(model: bestModel))
    }

    /// Find the smallest "Runs Well" model for this device.
    /// Falls back to nil (caller falls back to Browse) if none found.
    private func pickBestModel() -> DownloadedModel? {
        guard let engine = appContainer.compatibilityEngine else { return nil }
        return downloadedModels
            .filter { engine.compatibility(for: $0.repoId) == .runsWell }
            .min(by: { $0.fileSizeBytes < $1.fileSizeBytes })
    }

    // MARK: - MeshGradient background

    private var meshBackground: some View {
        // Copy exact MeshGradient implementation from BrowseView.swift or the
        // shared background component. The gradient colors are:
        // - Point [0,0]: #0A0918 (deep dark)
        // - Point [1,0]: #1E1B4B (deep indigo)
        // - Point [0,1]: #0F1729 (dark navy)
        // - Point [1,1]: #0D1520 (dark teal-black)
        // - Center: #16213E (midnight blue)
        // Animated with .animation(.easeInOut(duration: 8).repeatForever(autoreverses: true))
        // Read BrowseView.swift for the exact implementation — do NOT invent a new gradient.
        Rectangle()
            .fill(Color(hex: "#0A0918"))  // placeholder — replace with actual MeshGradient from BrowseView
            .ignoresSafeArea()
    }
}
